# Kissan Jam - India Codex'26 Submission

## Project Overview

Our project is a decentralized AI Services Marketplace built on Cardano. It connects AI providers with consumers, allowing seamless purchasing and utilization of AI services directly using ADA through Cardano smart contracts.

### Features
- Explore a marketplace of diverse AI models (NLP, Vision, Audio, Document AI, etc.)
- Connect with Cardano wallets using the Mesh SDK
- Seamlessly purchase AI usage tokens and invoke AI endpoints
- Smart contract escrow for trustless payments
